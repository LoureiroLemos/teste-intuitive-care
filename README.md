# Teste Técnico - Intuitive Care

Repositório criado para o desenvolvimento do teste técnico do processo seletivo de estágio da Intuitive Care.

O projeto está organizado por etapas, conforme descrito no enunciado do teste, separando integração com API, processamento de dados, SQL, backend e frontend.

A documentação detalhada de cada estapa será adicionada ao longo do desenvolvimento.
